# v5.5 Validation Checklist

Run:

```cmd
npm test
npm run validate
```

Then verify:

1. Open Trade Runs and confirm **My Runs** opens first.
2. Confirm My Runs displays the correct local run count.
3. Confirm the top Trade Runs badge matches the local run count.
4. Confirm there is no large blank or overlapping section above the saved runs.
5. Click **Examples 3** next to the Trade Runs heading.
6. Confirm the three dated examples appear in their own view.
7. Click **← My Runs** and confirm the personal-run list returns.
8. Copy an example and confirm it appears in My Runs while the example remains unchanged.
